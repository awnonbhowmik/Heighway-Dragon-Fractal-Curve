# Heighway Dragon Fractal Curve

A Python implementation of the [Heighway Dragon curve](https://en.wikipedia.org/wiki/Dragon_curve) using the `turtle` graphics module.

The Heighway Dragon (also known as the Harter–Heighway dragon) is a fractal curve that can be constructed by repeatedly folding a strip of paper in half and unfolding it so each fold is at a right angle. The resulting shape tiles the plane and has a fractal boundary.

## Prerequisites

- Python 3.x
- The `turtle` module (included in the Python standard library)

## Usage

```bash
python Heighway_Dragon_Fractal_Curve.py
```

A window will open and draw the dragon curve with green lines on a black background. Close the window to exit.

## Customization

In `Heighway_Dragon_Fractal_Curve.py`, the curve is generated by calling:

```python
dragoncurve(5, 15)
```

- **First argument (segment length):** Length of each line segment in pixels. Smaller values produce a more detailed curve.
- **Second argument (recursion depth):** Number of recursive iterations. Higher values produce a more complex curve but take longer to draw.

## Algorithm

The curve is generated using a recursive [Lindenmayer system](https://en.wikipedia.org/wiki/L-system) approach with two mutually recursive drawing functions (`x` and `y`) that alternate turning right and left at 90° angles.
