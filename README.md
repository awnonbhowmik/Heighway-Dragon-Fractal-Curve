# Heighway Dragon Fractal Curve

A Python implementation of the [Heighway Dragon curve](https://en.wikipedia.org/wiki/Dragon_curve) using the `turtle` graphics module.

## Requirements

- Python 3.x
- `turtle` module (included in the Python standard library)

## Run

```bash
python Heighway_Dragon_Fractal_Curve.py
```

A window will open and draw the dragon curve (green on black). Close the window to exit.

## Parameters

```python
dragoncurve(5, 15)
```

- `5` — segment length in pixels
- `15` — recursion depth (higher = more complex)
